Brand Overview

CentificAI aegis.ai is an AI-powered enterprise control centre. The design language is modern, clean, and trustworthy — balancing deep navy authority with vibrant purple energy and a signature magenta accent.

Brand Voice & Design Principles

🧠
Intelligent
The UI surfaces AI insights without visual clutter. Dense information is structured into scannable hierarchies with clear typographic contrast.
⚡
Responsive & Live
Real-time activity, live badges, and animation hints communicate a system that is always active and working on behalf of the user.
🔐
Trustworthy
Deep navy, structured grids, and precise status indicators convey reliability. Color is used intentionally — pink/purple accents signal primary actions, not noise.

Responsive Design

Breakpoints & Responsive Behaviour

The design system supports Desktop (1440px+), Laptop (1280px), and Tablet (768px–1024px). Mobile is currently out of scope. Key behaviours per breakpoint are listed below.

🖥
Desktop
1440px and above
  • Full 3-column card grid on home page
  • Two-column layout on inner pages (68/32 split)
  • All quick-action chips visible in 2 rows
  • Side activity panel always visible
  • Context banner full text visible
  • Form modals inline, full 2-col grid
💻
Laptop
1024px – 1440px
  • 3-column grid maintained with tighter gap (12px)
  • Inner page: 60/40 split (main/aside)
  • Header user name text may truncate
  • Hero headline scales to 32px
  • Prompt input max-width: 480px
  • Banner description text truncates to 1 line
📱
Tablet — Home
768px – 1024px
  • Card grid → 2 columns (below 900px → 1 column)
  • Hero headline reduces to 26px
  • Prompt input stretches full width (no max-width)
  • Quick-action chips reduce to 2–3 visible, rest hidden
  • Header: user name hidden, avatar only
  • Section header: search bar collapses to icon
📋
Tablet — Inner Page
768px – 1024px
  • Two-column collapses → single column
  • Activity panel becomes a collapsible drawer
  • Banner description hidden, title + actions remain
  • Form cards expand to full width
  • Chat input remains at bottom, full width
  • Activity drawer toggle button visible
